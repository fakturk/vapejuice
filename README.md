# vapejuice
Vape Juice rating system

## Purpose (Phase 1)
A website that users can rate and tag  vape juices

## Stacks

- Go
- GraphQL : [graphql-go](https://github.com/graph-gophers/graphql-go)
- ORM : [gorm](https://github.com/jinzhu/gorm)
